  1. Interpolation

    Getting variable value from name.component.ts by enclosing var name in between two nested flower brackets => {{interpolationExample}}

  2. Property Binding

    Variable values from name.component.ts can be binded to html properties using [property] = "varName"

    Interpolation can used as well to bind varNames to properties

    Example: [id] = varName

    When it comes to boolean values, interpolation doesn't work hence property binding should be used

    Boolean property example

    • [disabled] = "false"

    • [disabled] = "true"

  3. Class Binding

    Can be achieved by declaring
    varName = "css class-name" in name.component.ts
    [class] = "varName" in name.component.html

    Example:
    public classBindingExample = "text-success" in name.component.ts
    [class] = "classBindingExample" in name.component.html

    Test for class binding

    Boolean values in class binding

    Example:
    [class.css class-name] = "true" => Test for class binding

    [class.css class-name] = "false" => Test for class binding

    ngClass directive in class binding

    ngClass is mainly used to bind multiple CSS properties

    Example: ngClasses are defined in name.component.ts => Test for ngClass
